Добро пожаловать на главную страницу администратора!

На этой странице собрана информация о возможностях в рамках данной роли.

Администрирование. Регистрируясь в приложении как администратор, у пользователя появляется возможность создать группу для отслеживания чужих транзакций и планирования бюджета на основе общей статистики группы. При создании группы генерируется токен, его необходимо передать юзерам. Они используют его при регистрации.
При необходимости пользователя можно назначить ридером(читателем) для предоставления ему статистики по группе, но только для чтения.
Также администратор может не создавать группу и вести личный учет расходов и доходов.

Транзакции. Администратор может отслеживать общие расходы и доходы(если есть группа), а также свои собственные. Транзакции других пользователей администратор может изменять: редактировать описание и категорию.

Статистика. Для админа доступна общая статистика(по группе) и личная.

Добро пожаловать на главную страницу ридера!

На этой странице собрана информация о возможностях в рамках данной роли.

Доступ к данным пользователей. Ридер(или читатель) имеет полный доступ к информации о пользователях в группе, статистике и транзакциям, НО только для чтения. Это позволяет вести безопасную аналитику доходов и расходов, без внесения изменений.

Транзакции. Ридер имеет доступ к просмотру транзакций пользователей в группе, без реадктирования. Также, он может вести учет своих личных транзакций, при желании. Личные транзакции также отобразятся в статистике группы.

Статистика. Для ридера доступна общая статистика(по группе) и личная.

Добро пожаловать на главную страницу пользователя!

На этой странице собрана информация о возможностях в рамках данной роли.

Личные данные. Для пользователя доступны только его личные данные: транзакции и статистика по ним. Он может редактировать, удалять, добавлять и фильтровать их.

Транзакции. Внимательно добавляйте и редактируйте транзакции. Обновляйте свои расходы и доходы регулярно, тогда статистика будет более точной.

Статистика. Для юзера доступна только личная статистика по транзакциям.
